# IC UrFU Odoo Module

Кастомный модуль Odoo 17.0 для Уральского Федерального Университета (UrFU), который генерирует индивидуальные учебные планы для магистрантов.

## Возможности

- Генерация документов индивидуальных учебных планов в формате DOCX
- Соответствие официальным требованиям форматирования UrFU
- Поддержка обязательных и элективных дисциплин
- Интеграция с Odoo ERP framework

## Структура проекта

```
ic_urfu_module/
├── __init__.py              # Инициализация модуля
├── __manifest__.py          # Метаданные и зависимости
├── models/                  # Модели Odoo (ORM)
├── views/                   # XML определения представлений
├── doc-generator/           # Логика генерации документов
└── security/                # Контроль доступа
```

## Локальная разработка

### Требования

- Docker
- Docker Compose

### Запуск окружения разработки

```bash
cd local-env
docker-compose up -d
```

- **Odoo**: http://localhost:8069
- **PostgreSQL**: localhost:5432 (user: odoo, password: odoo, database: odoo)

### Остановка окружения

```bash
cd local-env
docker-compose down
```

### Тестирование генератора документов

Генератор документов можно запустить отдельно от Odoo:

```bash
cd ic_urfu_module/doc-generator
python doc-generator.py
```

Это создаст тестовый документ с примерными данными.
